## EFI介绍

此EFI引导适用于微星Z370 Krait gaming主板,MacOS Monterey,Opencore版本:0.7.4

## 可正常工作

- [x] 声卡（板载）/ 网卡（板载）
- [x] 显卡（核显 + 独显）/ 硬解 4K（HEVC + H.264）
- [x] WiFi（PCI-E 设备） / 蓝牙（PEI-E 载 USB 设备）
- [ ] 隔空投送 / 接力 / 随航(需白果卡)
- [x] FaceTime / iMessage
- [x] Apple Music / Apple TV Plus
- [x] 睿频 / HWP 变频 / 原生电源管理
- [x] 睡眠 / 键盘、鼠标唤醒
- [x] 定制USB,开启后置所有USB3/USB2,以及type-c
- [x] 其他白果功能（99%）
